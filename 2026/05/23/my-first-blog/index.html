我的第一篇 GitHub 博客：人生阶段的记录与思考

有些经历，当时只是普通的一天；
但多年以后再回头看，可能正是它们悄悄改变了自己。

写在前面

这是我在 GitHub Pages 上搭建的第一篇博客。

一开始想写博客的时候，我也想过很多方向：技术笔记、科研记录、论文阅读、项目总结。后来慢慢觉得，真正值得长期记录的，不只是“我学会了什么”，还有“我经历了什么”“我为什么会这样想”“我在哪些地方反复跌倒，又是怎样一点点调整自己的”。

所以，这个博客对我来说，不只是一个展示学习成果的地方，更像是一份长期的成长记录。

它会记录我在人生不同阶段里遇到的人、经历的事、犯过的错、想明白的道理，以及那些暂时还没有答案的问题。

为什么想开始写博客

人在不同阶段会经历很多事情。

有些是学习和工作中的问题，有些是生活里的小事，有些是和别人交流之后产生的触动，也有些是自己反复犯过的错误。

很多事情刚发生的时候，好像只是一个很普通的瞬间。但过一段时间再回头看，会发现它们其实影响了自己的想法、习惯和选择。

有时候，一些错误不是第一次出现。

明明之前已经遇到过类似的问题，换了一个场景之后，还是会以差不多的方式再次发生。

比如：

  • 明知道拖延会带来压力，却还是习惯等到最后才开始；
  • 明知道沟通很重要，却还是经常把问题先憋在心里；
  • 明知道行动比空想重要，却还是容易在开始前想太多；
  • 明知道有些事情没有那么严重，却还是会被焦虑困住；
  • 明知道自己已经经历过类似问题，却还是会重复同样的反应。

如果不记录下来，这些经历很容易被时间冲淡。下一次遇到类似情况时，可能还是会重复原来的思维方式和行为模式。

写下来，不一定能立刻改变什么。

但至少可以让自己看见问题。

我想记录什么

这个博客以后大概会记录下面几类内容。

01｜成长阶段中的体验和感悟

有些经历不一定很宏大，可能只是一次普通的聊天、一次活动、一段旅程，或者某个瞬间突然产生的想法。

但这些看似普通的片段，往往会在后来变成理解自己的线索。

我希望把这些时刻记录下来。

不是为了证明自己有多成熟，而是为了留下真实的变化过程。

02｜自己反复犯过的错误

有些错误会重复出现。

如果每次只是后悔一下，然后很快忘掉，下一次可能还是会以相似的方式发生。

所以我希望把这些问题写下来：

  • 当时为什么会这样做？
  • 后来发现哪里不对？
  • 这个问题以前是否出现过？
  • 下次遇到类似情况时，可以怎样调整？

写下来不是为了否定自己，而是为了更清楚地看见自己。

有时候，真正的成长不是一下子变得多厉害，而是开始意识到：

原来我总是在类似的问题上摔倒。

03｜学习、科研和实践中的思考

人生阶段里也离不开学习和实践。

无论是课程、科研、项目，还是某个技术工具的使用过程，都会有很多值得记录的地方。

很多问题解决之后，如果不记录下来，下次很可能还会重新踩坑。

所以我也会把一些学习过程、工具经验和项目总结整理到博客里。

这些内容可能包括：

  • GitHub 和博客搭建记录；
  • 编程、LaTeX、论文写作等工具经验；
  • 科研项目中的问题和思考；
  • 阅读论文后的理解与总结；
  • 项目推进过程中的踩坑和复盘。

这些内容不一定都是完整成熟的成果，也可以是某个阶段的想法、尝试和反思。

04｜生活中有意思的人和事

生活里会遇到很多有意思的瞬间。

有时候是一件小事，让我突然意识到自己以前的想法太片面。

有时候是一段经历，让我发现不同人看待世界的方式真的很不一样。

有时候是一次选择，让我重新理解什么是重要的，什么其实没有那么重要。

这些瞬间如果不写下来，很快就会被新的事情覆盖掉。

但它们其实也是一个人慢慢成长的痕迹。

写博客的意义

我觉得写博客最大的意义，是把“经历过”变成“真正理解过”。

很多时候，我们只是经历了一件事，但没有真正停下来想清楚。

等以后再遇到类似问题时，又会重复原来的情绪、判断和行为方式。

如果能把问题、过程、原因和感受记录下来，不一定能立刻改变什么，但至少可以让自己更清楚地看见自己。

博客对我来说，不只是一个展示结果的地方，更像是一个长期复盘和整理的空间。

它可以记录成功，也可以记录失败；
可以记录清楚的答案，也可以记录暂时想不明白的问题；
可以记录学习和科研，也可以记录生活、情绪和成长。

写给以后的自己

这是第一篇博客，也是一个开始。

后面我会尝试持续更新。内容不追求完美，但希望每一篇都能记录一个真实的问题、一段真实的经历，或者一个当下真实的想法。

也许几年之后再回头看，会觉得现在的很多想法还不成熟。

也许会发现，有些问题已经慢慢解决了。

也许也会发现，有些错误仍然在重复出现。

但这些都没有关系。

能看见自己的变化，本身就是一种成长。

结语

希望这个博客能成为我人生阶段中的一份长期记录。

记录那些经历过的事，犯过的错，想明白的道理，以及还在慢慢寻找答案的自己。

不是所有经历都会立刻给出答案。
但只要愿意记录，它们总会在某个时刻变成理解自己的线索。
